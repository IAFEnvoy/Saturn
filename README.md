# Saturn

Saturn is a lightweight x-plat API. Aimed at providing same APIs cross loaders and MC versions.

## Supported Versions

- Forge 1.19~1.20.1
- Fabric 1.19+
- NeoForge 1.21+

## Current Completed APIs

- Registry APIs
- Renderer APIs
- Menu&Screen APIs
- Key Mappings APIs
- Resource Reloader
- Partly Events
- Some Miscellaneous APIs

## Planned

- All Events
- Other Miscellaneous Registry APIs (e.g. BiomeModification, ItemProperties, DefaultEntityAttributes, RenderTypes,
  CreativeTabs)
